Selenium

Selenium — популярний інструмент для функціонального тестування веб-застосунків

Один з найбільш використовуваних інструментів для автоматизації веб-тестів. Підтримує різні мови програмування, такі як Java, Python, C#, Ruby, і може працювати з усіма популярними браузерами (Chrome, Firefox, Safari, Internet Explorer)

Офіційна сторінка

Cypress

Cypress — інструмент для проведення функціонального тестування веб-застосунків

Інструмент для тестування веб-додатків, який працює прямо в браузері. Має швидке налаштування, інтуїтивно зрозумілий інтерфейс та підтримує тестування з JavaScript.

Офіційна сторінка

Appium

Appium — інструмент для тестування мобільних застосунків

Підтримує автоматизацію тестів для мобільних застосунків на платформах Android і iOS. Використовує різні мови програмування, такі як Java, Python, JavaScript.

Офіційна сторінка

JMeter

JMeter — інструмент для проведення навантажувального та продуктивного тестування

Один із найпопулярніших інструментів для навантажувального тестування, підтримує тестування веб-застосунків, баз даних, FTP та інших протоколів. Має потужний інтерфейс для налаштування сценаріїв тестів.

Офіційна сторінка

Postman

Postman — інструмент для тестування API

Інструмент для тестування RESTful API, який дозволяє створювати запити, перевіряти відповіді сервера та автоматизувати тести за допомогою JavaScript. Має зручний інтерфейс для роботи з API.

Офіційна сторінка

Jenkins

Jenkins — один із найпопулярніших інструментів для автоматизації безперервної інтеграції (CI) і безперервного розгортання (CD). Має велику кількість плагінів і підтримує інтеграцію з різними інструментами.

Офіційна сторінка

Kubernetes

Kubernetes - популярна система оркестрації контейнерів для автоматизації розгортання, масштабування і управління контейнеризованими додатками.

Офіційна сторінка

Ansible

Ansible — потужний інструмент для автоматизації управління конфігурацією, оркестрації та обробки середовищ, який використовує YAML для написання сценаріїв.

Офіційна сторінка

Terraform

Terraform — Інструмент для автоматизації управління інфраструктурою як кодом. Підтримує різні хмарні платформи та забезпечує масштабованість і контроль над інфраструктурою.
