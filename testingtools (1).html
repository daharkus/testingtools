Apache JMeter

Apache JMeter — це популярний інструмент з відкритим вихідним кодом для тестування навантаження, продуктивності та функціональності веб-додатків.

JMeter підтримує різні типи тестів, зокрема тестування навантаження, стрес-тестування, тестування продуктивності, а також функціональні тести для вебсайтів та вебсервісів.

Gatling

Gatling — це популярний інструмент з відкритим вихідним кодом для тестування навантаження та продуктивності, орієнтований на веб-додатки.

Gatling дозволяє проводити стрес-тестування, навантажувальні тести та тести на продуктивність для перевірки стабільності систем під високим навантаженням.

Офіційна сторінка

Burp Suite

Burp Suite — це популярний набір інструментів для тестування безпеки веб-додатків. Він надає потужний набір функцій для виявлення вразливостей і проведення пентестів (тестів на проникнення) на веб-сайтах та веб-сервісах.

Burp Suite дозволяє виявляти різні типи вразливостей, зокрема SQL-ін'єкції, XSS (міжсайтові скриптові атаки), порушення контролю доступу та інші проблеми безпеки.

Офіційна сторінка

JUnit 5

JUnit — це фреймворк для модульного тестування в мові програмування Java. Він є одним з найбільш популярних інструментів для тестування програмного коду в Java-середовищі і дозволяє автоматизувати процес тестування окремих одиниць коду (методів або класів), що забезпечує їх правильність і стабільність.

Офіційна сторінка

WAWE

WAVE (Web Accessibility Evaluation Tool) — це інструмент для оцінки доступності веб-сторінок, розроблений компанією WebAIM (Web Accessibility In Mind).

Основна мета WAWE — допомогти веб-розробникам, дизайнерам та тестувальникам у виявленні проблем із доступністю веб-сайтів для людей з обмеженими можливостями.

Офіційна сторінка

Docker

Docker — це платформа для автоматизації розгортання, масштабування та управління контейнеризованими додатками. Вона дозволяє розробникам упаковувати програмне забезпечення в контейнери, що є легкими, ізольованими середовищами, які можуть запускатися на будь-якому сервері, що підтримує Docker, без необхідності враховувати залежності операційної системи або інші компоненти.
